Задания
API
Используя предоставленные данные CSV, создайте маршрут API с помощью Laravel, который позволит выполнять поиск данных.
Данные должны быть преобразованы в таблицу базы данных. Обеспечьте миграцию Laravel и сидеры в рамках проекта.
API должен выполнять поиск по:
     • Имя: также должно совпадать с частичными именами.
     • Спальни: точное соответствие
     • Ванные комнаты: точное соответствие
     • Этажность: точное соответствие
     • Гаражи: точное совпадение
     • Цена: диапазон (от X до Y долларов).
 
Все параметры поиска должны быть необязательными, мы должны иметь возможность искать дома с 2 спальнями или дома с 4 спальнями и 2 ванными комнатами и т. д.
 
API должен возвращать JSON с чистыми числовыми данными (не содержимым HTML).
 
Интерфейс (форма поиска)
 
Создайте простую форму поиска, которая будет отправлять AJAX-запросы к API и отображать полученные результаты во внешнем интерфейсе. Результаты поиска должны динамически отображаться в виде HTML-таблицы с использованием jQuery для обновления содержимого.

Включите на страницу индикатор загрузки, например вращающийся значок или аналогичный элемент, чтобы указать, что поиск выполняется.

Если результаты поиска не найдены, отобразите сообщение об отсутствии результатов.
